  • How to Begin Using GiftPrime?
In your WordPress plugin section, search for ‘GiftPrime’, then follow the instructions to get started. If you enjoy using GiftPrime and wish to upgrade, you can do so directly from the GiftPrime plugin dashboard.
  • Is GiftPrime compatible with all WordPress themes and page builders?
Yes, GiftPrime is designed to work seamlessly with any WordPress theme or page builder available in the market, ensuring a smooth and hassle-free integration with your existing website.
  • Do I need an Amazon API to use GiftPrime?
No, there’s no need for an Amazon API to use GiftPrime. All you need is a WordPress site and an Amazon affiliate tag from your account to get started.
  • What languages does GiftPrime support?
As Version 1.0.0, GiftPrime primarily supports English for content generation. We are continually working to expand our language capabilities to cater to a wider audience in the future.
  • What Amazon Stores does GiftPrime support?
As Version 1.0.0, GiftPrime supports Amazon.com And Amazon.co.uk
